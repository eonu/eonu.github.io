Formatting

Typora is a markdown editor which can handle and render LaTeX on the go, without requiring manual compilation.

This software was used to write these notes, and I strongly recommend it to anyone looking for a reliable tool for making notes and for general text processing.

Bug fixes

Sadly these notes are far from perfect—but you can help out by reporting any errors or suggestions through the GitHub repository.

Note: There are specific steps that must be done to successfully update a markdown document—please carefully follow the README instructions!
