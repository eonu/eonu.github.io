.. _validations:

Validations
===========

The :class:`_Validator` class is used internally for validating parameters of transformation classes.

This interface is only necessary to use if you are creating your own `custom transformations <https://nbviewer.jupyter.org/github/eonu/sigment/blob/master/notebooks/Custom%20Transformations.ipynb>`_.
